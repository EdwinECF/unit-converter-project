# Unit Converter

A responsive web application that converts common units of measurement with a simple and intuitive interface. Users can enter a value and instantly view conversions for length, volume, and mass.

## Features

* Convert meters to feet and feet to meters
* Convert liters to gallons and gallons to liters
* Convert kilograms to pounds and pounds to kilograms
* Perform instant calculations based on user input
* Clean and responsive user interface

## Built With

* HTML5
* CSS3
* JavaScript (ES6)
